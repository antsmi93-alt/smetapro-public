# Универсальный калькулятор смет для подрядчиков

## Что это
Приложение в кармане для мастера любой специальности. Стоя на объекте у клиента — открыл телефон, отметил галочками работы и количество, нажал итог — получил точную цифру.

## Где файл
- Локально: `Vault Антона/06-Production/Универсальный калькулятор/calculator-universal.html`
- GitHub: `antsmi93-alt/H_93_test_2/universal/calculator-universal.html`

## Как открыть
1. Перекинь файл на телефон через Telegram, AirDrop, Google Drive или iCloud
2. Открой в браузере (Chrome, Safari, Brave)
3. Работает полностью офлайн — ни одного запроса в интернет

## Готовые профили (из коробки)

| Профиль | Примеры работ |
|---------|---------------|
| ⚡ Электрик | Розетки, выключатели, кабель, щитки, люстры, штробление |
| 🔧 Сантехник | Смесители, унитазы, ванны, душевые, трубы, счётчики |
| 🔥 Газовщик | Плиты, колонки, котлы, перенос точки, датчики |
| 🔨 Отделочник | Шпатлёвка, поклейка обоев, плитка, ламинат, гипсокартон, двери |
| 🌬 Вентиляция/кондиционеры | Сплиты, воздуховоды, вентиляторы, чистка, дозаправка |

## Как пользоваться

### Новая смета (вкладка "➕ Новая смета")
1. Выбери профиль — профиль = твоя специальность
2. Введи имя клиента / адрес объекта — чтобы потом не забыть
3. Отметь галочками нужные работы → введи количество
4. Добавь материалы, если нужно (вручную или CSV)
5. Укажи скидку или наценку в процентах
6. Итог — точная цифра для клиента
7. Сохрани смету → она остаётся в телефоне

### Сохранённые сметы (вкладка "📁 Мои сметы")
- Все сохранённые сметы с датой и суммой
- Можно открыть и пересчитать
- Можно удалить

### Настройки профиля (вкладка "⚙ Профиль")
- **Редактировать** любой профиль: добавить/удалить работы, поменять цены
- **Создать новый профиль**: любая специальность — стекольщик, каменщик, сварщик
- **Загрузить CSV**: массово загрузить материалы
- **Экспорт/импорт JSON**: перенести всю базу на другой телефон

## Как добавить свой профиль

1. Вкладка "⚙ Профиль" → нажми "+ Новый"
2. Введи название профиля (например: "🔩 Сварщик")
3. В текстовое поле "Работы" вставь список в формате:
```
Сварка металла 3 мм;450;п.м
Сварка металла 5 мм;650;п.м
Резка УШМ;200;п.м
Покраска металла;350;м2
```
4. Нажми "💾 Сохранить профиль"
5. Переходи на "➕ Новая смета" — твой профиль уже в списке

## Формат CSV для загрузки материалов

```csv
Название;Цена;Единица
Кабель ВВГ 3x1.5;45;п.м
Розетка Legrand;280;шт
Гофра 16 мм;25;п.м
```

## Как поделиться сметой с клиентом

1. Нажми "📤 Поделиться" → сработает встроенная функция телефона (WhatsApp, Telegram, SMS)
2. Или нажми "🖨 Печать" → сформируется красивый текст для печати / PDF

## Советы по ценообразованию

- Цены в дефолтных профилях — усреднённые по рынку 2026
- **Обязательно** перепиши цены под свой регион. Москва и регион — разница в 2-3 раза
- Если работа не в списке — добавь в профиль за 10 секунд

## Почему именно HTML-файл

- Работает везде: iPhone, Android, Windows Phone, компьютер
- Не нужно скачивать из App Store / Google Play
- Не нужен интернет после загрузки
- Данные хранятся в телефоне — никуда не утекают
- Можно скинуть файл другому мастеру — и у него тоже заработает

## Установка как PWA (Progressive Web App)

В версии 1.1.0+ калькулятор можно установить как обычное приложение — прямо с сайта, без App Store.

**На iPhone/iPad (Safari):**
1. Открой страницу калькулятора в Safari
2. Нажми кнопку «Поделиться» (квадрат со стрелкой)
3. Пролистай вниз → «На экран Домой»
4. Подтверди имя → «Добавить»
5. Иконка «Смета PRO» появится на рабочем столе

**На Android (Chrome):**
1. Открой страницу в Chrome
2. Появится плашка «Установить приложение» — нажми
3. Или: меню Chrome (⋮) → «Добавить на главный экран» / «Установить приложение»
4. Иконка появится среди приложений

**Что даёт PWA-установка:**
- Запускается как обычное приложение, без адресной строки
- Работает офлайн полностью (Service Worker кеширует всё)
- Занимает ~200 КБ — это меньше, чем одна фотка в галерее
- Обновляется автоматически при выходе новой версии

## Перенос на другой телефон

1. Вкладка "⚙ Профиль" → нажми "📥 Сохранить базу JSON"
2. Файл скачается
3. Перекинь файл на новый телефон
4. На новом телефоне открой калькулятор → "📤 Загрузить базу JSON" → выбери файл
5. Все сметы и профили перенесены
